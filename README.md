# PageRank
This project is an attempt to construct the foundation of the Google PageRank algorithm and practice how to apply linear algebra concepts to real-world solutions.

## Folder Descriptions
- Code & Execution: Contains the notebook file with the code and the pdf file with the notebook execution
- Presentation: Contains the slides explaining our implementation of the Google PageRank algorithm
